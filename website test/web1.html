歷史
1980年，物理學家提姆·柏內茲-李在歐洲核子研究中心（CERN）在承包工程期間，為使CERN的研究人員使用並共享文件，他提出並建立原型系統ENQUIRE。1989年，柏內茲-李在一份備忘錄中提出一個基於網際網路的超文字系統。他制定HTML並在1990年底寫出瀏覽器和伺服器軟體。同年，柏內茲-李與CERN的資料系統工程師羅伯特·卡里奧聯合為專案申請資助，但未被CERN正式批准。在他的個人筆記中柏內茲-李列舉「一些使用超文字的領域」，並把百科全書列為第一項。 HTML的首次公開描述出現於一個名為HTML Tags的檔案中，由提姆·柏內茲-李於1991年底提及。它描述18個元素，包括HTML初始的、相對簡單的設計。

HTML早期標誌
HTML是甚麼?
HTML(Hyper Text Markup Language)是一種標記式語言，是一種描述超文件的註記語言SGML(Standard Generalized Markup Language)所制訂出的一種網頁語言，基本上現行的瀏覽器都可以讀取HTML，使用HTML可以編輯設計出網頁，也可以在網頁中加入所有HTML語言可支援的方式，例如表格、表單、圖片、文字、連結、程式等等。 HTML是一種非常簡單的語言，由一系列元素所組成。元素可以賦予文字片段不同的意義、將文件組織成不同的邏輯區段，以及在網頁中嵌入圖片或影片等內容。
HTML基礎概念(怎麼使用)
  • 元素（elements）
    • 在 HTML 中，元素可能包含資料清單、文字、圖片，或是也可能不包含東西。典型的元素會包含一個有些許屬性（attributes）的開始標籤（opening tag）、被包覆住的文字內容，以及一個結束標籤（closing tag）。
  • 標籤 (Tag)
    • 在原始碼中，標籤(使用角括號 < > 包裹住的文字)用來標注元素開始與結束。
  • 起始標籤（opening tag）
    • 它包含了元素的名字（在這裡是 p），夾在一對打開和關閉尖括號之間。它指明元素從何開始生效──在上例中則代表段落的開始。
  • 結束標籤（closing tag）
    • 結束標籤和起始標籤長得差不多，只不過它在名字前面還多加了一條斜線（forward slash）。它表示元素結束的地方──在上例中表示該段落的結束。
  • 功能性標籤
    • 標籤本身不需要內文。
  • 內文
    • 內文可以是純文字，或包含其他標籤。 屬性（attributes） 標籤的附加資訊，協助標籤運作。
  • 語法
    • 屬性名稱="屬性的內容" 屬性包在起始標籤裡面，如範例所示。 在元素名稱和屬性之間有一個空格（如果有多個屬性，屬性之間也需要有空格）。 屬性名稱後面接著等於符號「=」。
  • 註解
    • 使用兩組符號將註解包裹起來: ＜!-- 註解內文打這裡 --＞
  • HTML基礎網頁結構
  • ＜!DOCTYPE html＞
    • 文件類型（doctype）。 在 HTML 發展初期（約莫 1991/2 的年代）文件類型是用來連結一些應遵守的規則，有點像自動校正的功能。
  • ＜html＞＜/html＞
    • ＜html＞ 元素，又被視為根元素（root element），包含了所有顯示在這個頁面上的內容。
  • ＜head＞＜／html＞
    • ＜head＞ 元素，裡面放的是你想涵括的重要資訊，但不會顯示於網頁瀏覽者眼前的。例如，顯示於搜尋結果的關鍵字、頁面說明、CSS、字元實體集...等。
  • ＜body＞＜／body＞
    • ＜body＞ 元素，包含了所有會顯示於網頁瀏覽者眼前的內容。 無論是文字、圖片、影面、互動遊戲...等。 這個元素指定了你的文件使用 utf-8 這種字元編碼，使用這個元素，會免去許多文字無法正確呈現的煩惱。
  • ＜title＞＜/title＞
    • 呈現於網頁瀏覽者眼前的網頁標題。
  • 標記文字
    • 文件標題(heading):
    就像一本書有書名、章節名稱和副標題，一份 HTML 文件也有類似的概念。HTML 最多可以有六層的 heading。
    段落:
    ＜p＞elements 包含文字段落，在呈現一般文字時，是最常用到的。
    清單:
    使用 ＜ul＞和 ＜li＞ 呈現。清單至少會包含兩個元素。
    連結:
    對網頁來說，連結是非常重要的。用這個元素讓文字變成連結。搭配 herf 屬性指定目標網頁位址。

  • 圖片
    • 使用＜img＞將圖片呈現在網頁中。 搭配 src 屬性指定圖片檔案位址， alt 屬性用於當網頁瀏覽者無法證確看到圖片時，會呈現出什麼樣的文字說明。
  • 表格
    • 使用 ＜table＞、＜tr＞、＜td＞ 建立表格。
