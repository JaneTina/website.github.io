JavaScript歷史
JavaScrpit 誕生於1995年，由 Netscape 這家公司專門為其 Netscape 瀏覽器而發明的。一開始的角色是讓使用者與網頁有即時的互動效果，例如檢查表單、跳出 Alert 視窗，以及網頁特效。
JavaScript是什麼?
JavaScript是一種腳本、事件驅動的程式語言，一般簡稱JS，它支援物件導向程式設計、指令編程以及函式語言程式設計，它讓網站中出現較為複雜的功能，比如動態內容、2D/3D圖形繪製、影片播放控制…等。簡單來說就是讓網站可以有特效、效果與使用者互動，比如像是網頁上的圖塊移動、文字淡出淡入或者是按鈕點擊效果等。
JavaScript基本概念
  • 變數 (Variable)
      • 變數可以代表著很多類別，這些類別，我們稱之為data type。常用的data type：
      Integer (數字)
      Float (有小數點的數字)
      String (文字) :
      當變數儲存的是文字時，則需要用一個單引號'或雙引號"包著要儲存的文字。
      Boolean(TRUE/FALSE)
  • 函數 (Function)
      • 編程時，當需要重複去寫一些同樣的東西時，為了避免浪費時間，便會使用函數 (Function)來編寫程式碼。
      Return
  • Compare Expression
      • == 等於
      != 不等於
      > 大於
      < 小於
      >= 大於或等於
      <= 小於或等於
      && and
  • Condition
      • 正常的程式會一句一句的去運行，但在某些情況下，我們只想運行某一部份的程式碼，這時我們就會需要用到condition。
    • If...else
    • else if
      • else if在同一個比較語句中，可以使用很多次，但else就只能使用一次。
  • Loop
      • 一些指令需要不斷重複去執行，我們便會使用Loop。
    • While loop
    • For loop
      • =0，是i的初始狀態。
      i< 10，運行不會大於10。
      i++，代表每過一個loop，i的數值都會加1。
  • Array(數組)
      • 意思是一組相似的數據，通常會使用[]去表達。
  • Object(物件)
      • 通常會用{}去表達。Object比Array更強大，因為它可以儲存一大組不同data type的數據，甚至是functions。
  • Callback Function
      • 代表當一個Function運行完後，接下來會運行另一個Function。
  • 處理String
      • substring
      indexof
  • Debug
